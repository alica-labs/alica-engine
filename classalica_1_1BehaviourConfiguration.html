Class alica::BehaviourConfiguration

A Behaviour Configuration encapsulates a set of static parameters and a set of variables for a (Basic) Behaviour .
The BehaviourConfiguration is indirectly derived from AlicaElement , therefore it is owned by the PlanRepository and should never be deleted or changed by anybody else.

Private Members

bool eventDriven
Specifies whether this Behaviour is run eventDriven. If it is not event driven, a timer will call it according to Frequency and Deferring.
int frequency
The frequency with which this Behaviour is called in case it is not EventDriven.
int deferring
The time in ms to wait before this Behaviour is executed for the first time after entering the corresponding state. Has only effect for Behaviours not running in EventDriven mode.
shared_ptr< map< string, string > > parameters
The set of static parameters of this Behaviour configuration. Usually parsed by BasicBehaviour.InitializeParameters.
const Behaviour * behaviour
This configuration's Behaviour
